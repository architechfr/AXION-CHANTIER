Comptes

Note

La création de nouveaux comptes (envoi d'invitation) nécessite le système d'invitation — prochaine étape. Ici on gère les comptes existants : rôle, nom et chantiers. Le rôle admin donne accès à tout ; moe/moa/entreprise/économiste ne voient que leurs chantiers rattachés.

Le Statut interne Cadence a deux dimensions orthogonales : le métier (Architecte · Maître d'œuvre · Économiste · Collaborateur · Stagiaire) et la qualité d'associé (case à cocher, valable pour architecte/MOE/économiste uniquement). Une personne peut donc être « Architecte associé », « Architecte » (salarié), ou « Stagiaire ». Pour l'activer en base, ajoutez une fois ces deux colonnes : alter table public.user_profiles add column if not exists statut text;
alter table public.user_profiles add column if not exists associe boolean not null default false;
